## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Dark Premium Theme** | Deep indigo/pink/cyan color palette with glassmorphism effects |
| 🌌 **Live Particle Background** | Canvas-based animated particles with dynamic connecting lines |
| 🖱️ **Custom Cursor** | Glowing dot + trailing ring with hover effects |
| ⌨️ **Typing Animation** | Auto-typing bio text with blinking cursor |
| 📜 **Scroll Reveals** | Intersection Observer-powered fade-in animations |
| 🔢 **Counter Animation** | Animated stat counters that count up when scrolled into view |
| 🪐 **Orbital Hero Visual** | Floating profile photo with animated gradient orbit rings |
| 📱 **Fully Responsive** | Mobile-first design with hamburger menu navigation |
| 🔊 **Noise Texture Overlay** | Subtle SVG noise for a premium tactile feel |
| ⚡ **Zero Dependencies** | Pure HTML, CSS & JavaScript — no frameworks needed |
